# 👤 Gender Classification Using Machine Learning

A Machine Learning web application built with **Flask** that predicts gender based on facial features using a trained classification model.

---

## 📌 Features

- Predict gender using Machine Learning
- Simple and user-friendly interface
- Fast predictions
- Built with Flask
- Responsive HTML & CSS design

---

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

---

## 📂 Project Structure

```
Gender-Classification-ML/
│── app.py
│── model.pkl
│── requirements.txt
│── templates/
│     └── index.html
│── static/
│── README.md
```

---

## 📊 Input Features

- Long Hair
- Forehead Width (cm)
- Forehead Height (cm)
- Nose Wide
- Nose Long
- Lips Thin
- Distance Nose to Lip Long

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Gender-Classification-ML.git
```

### Move into the project folder

```bash
cd Gender-Classification-ML
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```

---

## 🚀 How It Works

1. Enter all required facial feature values.
2. Click **Predict Gender**.
3. The Machine Learning model predicts the gender.
4. The result is displayed instantly.
